This section provides technical guidance and support for using the SHK website, including troubleshooting common issues and understanding website features.

Technical Support

If you encounter any technical issues while using the site, please try the following steps:

  • Clear your browser cache and cookies.
  • Ensure your browser is up to date.
  • Try accessing the site from a different browser or device.

Contacting Support

If the issue persists, please reach out to our support team via the Contact page with a detailed description of the problem.

Technical Feedback and Reporting

We welcome feedback on technical aspects of the site. If you have suggestions for improvements or wish to report bugs, please use the Feedback page to share your thoughts, or submit a quick response on page 3 of our Feedback form.

Website Features Overview

For a comprehensive guide on how to navigate and utilize the various features of the SHK website, please visit our main Help & Using the Site section.

Open Access Code & Data

Seeking His Kingdom is committed to transparency and open access. Our website resources, including codebase and datasets are available on GitHub.
